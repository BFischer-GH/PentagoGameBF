Groupings of Variable

How Do Arrays Function?

Let's start learning about arrays. Arrays are essentially groupings of variables. They allow multiple variables of the same type to be stored easily. There are two ways to make an array. We can initialize it and give it all the variables we need to store, or initialize it by specifying the length of the array. This is by far the more common method since it is more modular.

The first way of making an array is shown to you. Let's demonstrate the second way. Here is the syntax.int[] name = new int[length] The variable type can be changed to whatever you want.

Another important thing about arrays is to set and get values, we need to specify which part of the array to get and set from. Imagine that each value in the array is given a number from 0 to length-1. This is called index based, because the numbering starts at 0. Try making an integer array with the length of 3 using the second method. Then set the 3 values to whatever you want using: arrayName[index] = value;
